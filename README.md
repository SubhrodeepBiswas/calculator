# 🧮 Advanced Calculator App

A modern **Python Tkinter Calculator**, featuring multiple calculator modes, unit converters, graph plotting, date calculations, programmer functions, calculation history, and light/dark themes.

---

## ✨ Features

### 🖩 Standard Calculator
- Basic arithmetic operations
- Percentage calculations
- Square, square root, reciprocal
- Positive/negative toggle
- Backspace support

### 📐 Scientific Calculator
- Trigonometric functions
- Logarithms (`log`, `ln`)
- Powers and roots
- Factorials
- π and e constants
- Scientific notation

### 📈 Graphing Calculator
- Plot mathematical functions
- Supports:
  - `x**2`
  - `sin(x)`
  - `cos(x)`
  - `sqrt(x)`
  - and more

### 💻 Programmer Calculator
- Number systems:
  - Binary
  - Octal
  - Decimal
  - Hexadecimal
- Bitwise operations:
  - AND
  - OR
  - XOR
  - NOT
  - Left Shift

### 📅 Date Calculator
Calculate the difference between two dates in:

- Days
- Months
- Years

### 🔄 Unit Converters

Supports conversion of:

- 💱 Currency
- 📏 Length
- ⚖️ Weight & Mass
- 🌡 Temperature
- ⚡ Energy
- 🗺 Area
- 🚀 Speed
- ⏱ Time
- 🔌 Power
- 💾 Data
- 🔵 Pressure
- 📐 Angle
- 🧪 Volume

### 📜 History Panel
- Stores previous calculations
- Scrollable history view

### 🌙 Dark Mode
- Toggle between:
  - Light Theme
  - Dark Theme


Move into the project directory:

```bash

```

Run the program:

```bash
python calculator.py
```
